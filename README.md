## Project Overview
This project visualizes 6 Uninformed Search algorithms on a 10x10 grid. It demonstrates how different "blind" strategies find a path from Start (S) to Target (T).

## Algorithms Implemented:
1. BFS (Breadth-First Search)
2. DFS (Depth-First Search)
3. UCS (Uniform-Cost Search)
4. DLS (Depth-Limited Search)
5. IDDFS (Iterative Deepening DFS)
6. Bidirectional Search

## Features:
- **Movement Order:** Strict clockwise order (Up, Right, Bottom, Bottom-Right, Left, Top-Left).
- **Visualization:** Clearly shows Frontier (Orange), Explored (Cyan), and Final Path (Yellow).
- **Dynamic Obstacles:** Randomly appearing walls to test agent re-planning.

## How to Run:
